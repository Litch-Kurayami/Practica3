< html >

La eriqueta "< html >" es el elmento raíz de un documento HTML y envuelve todo el contenido de la página web. Indica al navegador web que el documento es un documento HTML y que todo el contenido debe ser interpretado como tal. Esta etiqueta debe estar presente alrededor de todo el contenido de la página.

< p >< !DOCTYPE html > < html lang="es" > < head > < meta charset="UTF-8"> < meta name="description" content="Ejemplo de página web"> < title>Mi Página Web < /head> < body> < h1>Bienvenido a mi página web < p>Esta es una página de ejemplo que utiliza la etiqueta < ;html> ; como elemento raíz.< /p> < /body> < /html>< /p>

Null

< head >

La etiqueta < head > proporciona información meta sobre el documento HTML. Contiene elementos como metadatos, enlaces a archivos CSS y JavaScript, enlaces a fuentes, definiciones de estilos, y más. El contenido dentro de la etiqueta < head > no es visible para los usuarios en la página web, pero es crucial para el correcto funcionamiento y la optimización de la página.

< head > < meta charset="UTF-8" > < meta name="description" content="Ejemplo de página web" > < meta name="keywords" content="HTML, tutorial, ejemplo" > < meta name="author" content="Tu Nombre" > < title>Mi Página Web< /title > < link rel="stylesheet" href="estilos.css" > < /head >

Null

< title >

La etiqueta < title > se utiliza dentro del elemento < head > para definir el título de la página web. Este título se muestra en la barra de título del navegador y es importante tanto para la experiencia del usuario como para la optimización de motores de búsqueda (SEO). El título de la página debe ser descriptivo y relevante para el contenido de la página.

< title >Mi Página Web - Inicio< /title >

Null

< div >

La etiqueta < div > se utiliza para crear divisiones o contenedores en una página web. Se utiliza para agrupar y estructurar contenido relacionado, lo que facilita la aplicación de estilos CSS y el manejo del diseño de la página.

< div class="container" > < h1>Título de la página < p>Este es un párrafo de texto introductorio. < ul > < li >Elemento de lista 1< /li > < li >Elemento de lista 2< /li > < li >Elemento de lista 3< /li > < /ul > < /div >

< h1 >

La etiqueta < h1 > se utiliza para definir el encabezado principal o el título de una sección de contenido. Es el nivel más alto de encabezado y generalmente se usa para el título principal de la página o de una sección importante.

< h1 >Título de la página< /h1 >

< p >

La etiqueta < p > se utiliza para definir un párrafo de texto en una página web. Se utiliza para envolver y estructurar bloques de texto para que sean más legibles y estén separados visualmente del resto del contenido.

< p >Este es un párrafo de texto introductorio.< /p >

< ul >

La etiqueta < ul > se utiliza para crear una lista no ordenada en una página web. Cada elemento de la lista se define utilizando la etiqueta < li >, que se coloca dentro de la etiqueta

< ul > < li >Elemento de lista 1< /li > < li >Elemento de lista 2< /li > < li >Elemento de lista 3< /li > < /ul >

< li >

La etiqueta < li > se utiliza para definir un elemento de una lista (ya sea ordenada o no ordenada) en una página web. Estos elementos se pueden enumerar o presentar en viñetas, según el tipo de lista.

< li >Elemento de lista 1

< p >Este es un ejemplo de texto en < i >cursiva< /i > utilizando la etiqueta < ;i> ;.< /p>

< strong >

La etiqueta < strong > se utiliza para enfatizar el texto con un mayor nivel de importancia que < i >. Por lo general, se representa en negrita.

< title >Ejemplo de etiqueta < strong >< /title >

< section >

La etiqueta < section > se utiliza para definir una sección o área temática en un documento HTML. Ayuda a organizar y estructurar el contenido de la página.

< section > < article > < h2 >Artículo 1< /h2 > < p >< img src="imagen1.jpg" alt="Descripción de la imagen" >< /p > < p >Contenido del artículo 1< /p > < /article > < article > < h2 >Artículo 2< /h2 > < p >< video src="video.mp4" controls>< /video>< /p> < p >Contenido del artículo 2< /p > < /article > < /section >

< link >

La etiqueta < link > se utiliza para enlazar recursos externos, como hojas de estilo CSS o fuentes de caracteres , con el documento HTML.

< link rel="stylesheet" href="estilos.css" >

< header >

La etiqueta < header > se utiliza para definir el encabezado de una sección o el encabezado de la página web.

< header > < h1 >Título de la página< /h1 > < nav > < ul > < li >< a href="#">Inicio< /a >< /li > < li >< a href="#" >Acerca de< /a >< /li > < li >< a href="#">Contacto< /li > < /ul > < /nav > < /header >

< footer >

La etiqueta < footer > se utiliza para definir el pie de página de una sección o de la página web en general.

< footer > < p >Derechos de autor & copy; 2024 - Mi Sitio Web< /p > < /footer >

< Nav >

La etiqueta < nav > se utiliza para definir una sección de navegación, que contiene enlaces a otras páginas o secciones dentro del sitio web.

< nav > < ul > < li >< a href="#">Inicio< /a >< /li > < li >< a href="#" >Acerca de< /a >< /li > < li >< a href="#">Contacto< /li > < /ul > < /nav >

< article >

La etiqueta < article > se utiliza para definir contenido independiente y autónomo, como publicaciones de blog, noticias, comentarios, etc.

< article > < h2 >Artículo 1< /h2 > < p >< img src="imagen1.jpg" alt="Descripción de la imagen">< /p > < p >Contenido del artículo 1< /p >

< img >

La etiqueta < img > se utiliza para insertar imágenes en una página web.

< p >< img src="imagen1.jpg" alt="Descripción de la imagen" >< /p >

< video >

La etiqueta < video > se utiliza para insertar contenido de vídeo en una página web.

< p >< video src="video.mp4" controls >< /video >< /p >

< audio >

La etiqueta < audio > se utiliza para insertar contenido de audio en una página web.

< audio controls > < source src="cancion.mp3" type="audio/mpeg" > < /audio >
